# Visual Language from Vault

Observed recurring traits in the saved Ship Sh!t Show thumbnails:

## Composition

- hosts often live on the far left and far right
- center of frame is dominated by one oversized symbolic object
- backgrounds are dark, with the center light doing most of the storytelling
- compositions stay simple enough to parse instantly

## Faces and gestures

- strong facial expressions
- hands often reinforce the story
- one emotion per face, not mixed micro-expressions
- reactions are exaggerated enough for thumbnail scale

## Lighting

- hot orange-red glow is common
- blue glow is used effectively in comparison or tech-control setups
- the center object often acts as the primary light source

## Object language

- icons, vaults, chains, cracked objects, broken shells, warning symbols
- object is not decorative; it is the story

## Practical rule

If the user supplies a dense cinematic prompt, keep the cinematic energy but simplify the number of competing visual beats until the image reads in one glance.

## When to deviate

Only break from this language when:

- the user explicitly asks for a reset
- the episode topic clearly requires a different center symbol
- a more minimal composition improves small-size readability
