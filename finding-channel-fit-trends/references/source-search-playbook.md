# Source Search Playbook

Use live browsing for this skill.

## Reddit

Look for:

- fast-moving AI threads
- model comparisons
- security incidents
- agent tooling launches
- dev workflow pain points

Useful search angles:

- `site:reddit.com AI agents security`
- `site:reddit.com Claude vs GPT coding`
- `site:reddit.com MCP tool launch`
- source-specific searches for subreddits when needed

## Hacker News

Look for:

- launch posts
- security incidents
- benchmark debates
- devtools that trigger strong technical discussion

Useful angles:

- newest and front page AI/devtools stories
- comments with high engagement around AI coding tools, agent tooling, and security claims

## YouTube

Look for:

- repeated topic spikes across creator channels
- thumbnails and titles that keep recurring
- newly rising narratives around the same product or event

Useful angles:

- `<topic> YouTube`
- `<model> vs <model> YouTube`
- `<product> security YouTube`

## X

Look for:

- announcement tweets
- explosive benchmark screenshots
- founder or researcher callouts
- leak discourse
- controversy with obvious replay value

Useful angles:

- entity-name plus the topic
- newest posts around the event
- quote-tweet debates when accessible

## Cross-source rule

Do not stop at one platform.

A good brief usually has:

- one origin or breakout source
- one or more validation sources
- one indication that the topic can travel into video
