# Thumbnail Style Families

Choose one family first, then write variations inside that family or across adjacent families.

## 1. Threat / breach / too powerful

Use for:

- security failures
- leaks
- restricted models
- "something got out"
- dangerous capability stories

Visual defaults:

- red-orange eruption
- warning symbols
- cracked containment object
- blast lighting from center

Center object ideas:

- cracked model icon
- broken vault
- breached shield
- leaking containment chamber

## 2. Benchmark / versus / verdict

Use for:

- model comparisons
- tool comparisons
- best-model verdicts
- battle framing

Visual defaults:

- split-color clash
- impact collision
- shattered midpoint
- two-color energy contrast

Center object ideas:

- colliding logos
- split scoreboard-like symbol
- fractured shared icon

## 3. Product reveal / control / scanning

Use for:

- demos
- computer control
- surveillance or active scanning
- "it sees everything"
- emerging capability reveals

Visual defaults:

- scanning beam
- central device glow
- target overlays
- strong directional light

Center object ideas:

- scanning model icon
- controlled monitor
- activated terminal cloud
- radar-style lock-on motif

## 4. Build / demo / tutorial

Use for:

- installs
- walkthroughs
- rebuilds
- setup and onboarding

Visual defaults:

- one clear tool or system object in center
- strong action cue
- less chaos than threat mode
- still bold enough for YouTube

Center object ideas:

- app stack
- installation pipeline
- orchestration dashboard
- hardware or software build artifact

## 5. Policy / restriction / secrecy

Use for:

- blocked releases
- withheld access
- private programs
- closed betas
- internal-only systems

Visual defaults:

- chains
- vault doors
- locks
- warning triangles
- light leaking through narrow gaps

Center object ideas:

- chained icon
- sealed vault
- locked terminal
- hidden glowing core

## Global composition rules

- Keep one dominant center object.
- Put hosts on the outer edges when using host-led compositions.
- Use the light source to explain the narrative.
- Favor large shapes over fine detail.
- Keep the image readable with no text at 120px.
